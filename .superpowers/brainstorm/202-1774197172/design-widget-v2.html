✅ Widget Features
  • Floating bubble in corner (customizable position)
  • Notification badge for new messages
  • Smooth open/close animation
  • Real-time typing indicators
  • Source citations from documents
  • File attachment support
  • Customizable colors (header, messages, buttons)
  • Mobile-responsive design
  • Clean SVG icons (Lucide icon set)
🎨 Customization Options
  • Colors: Header, bubble, user messages, buttons
  • Position: X/Y offset, corner selection
  • Behavior: Auto-open delay, exit intent trigger
  • Greeting: Custom welcome message
  • Bot name: Display name in header
  • Avatar: Upload custom image
  • Size: Widget width/height options
